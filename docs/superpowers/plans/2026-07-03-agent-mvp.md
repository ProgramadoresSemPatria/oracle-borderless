# Oracle Agent MVP (v1) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the conversational oracle MVP — a chat that answers questions grounded exclusively in approved Notion docs (RAG over pgvector) + web search, always citing sources, streamed over SSE, with a provider-agnostic (Claude/GPT) Pydantic AI engine.

**Architecture:** Spatie-like layering (`app → domain → support`). The Pydantic AI engine lives in `src/support/agent/` behind a thin port; domain Actions (`conversations/`, `documents/`) consume it without importing the framework. Hybrid retrieval: semantic search over `document_chunks` (pgvector) + on-demand MCP `fetch_notion_page`. Answers stream via a pure-ASGI (non-buffering) DB-session middleware.

**Tech Stack:** FastAPI, SQLAlchemy 2.0 async, Alembic, Pydantic v2, pgvector, Pydantic AI, OpenAI embeddings (`text-embedding-3-small`, 1536d), Tavily web search, APScheduler.

## Global Constraints

- **Layering (regra 1):** `src/domain/{ctx}/entities/` must NOT import `sqlalchemy`, `fastapi`, or `pydantic`. Models/Actions/Services may import SQLAlchemy but never HTTP.
- **Entity ≠ Model (regra 2):** Entities are pure dataclasses; Models are SQLAlchemy. Conversion lives ONLY in the subdomain `mappers/`, never inline in repositories.
- **Session from context (regra 3):** repositories read the session via `CurrentAsyncSessionContext.get()`; never construct `AsyncSessionLocal()` in a controller/action/repository. (Exception introduced by this plan: the SSE middleware itself manages the session — see Task 19 / ADR-0010.)
- **Approved-only knowledge (regra 4):** only `status == "approved"` documents may be chunked, embedded, indexed, retrieved, or cited. Nothing confidential enters the base.
- **Thin controllers (regra 5):** controller extracts request → calls ONE action → returns a Response schema. No business logic.
- **No aggregator Service (regra 6):** one Action per use case; Actions call Actions to compose. Domain Services only for owner-less logic (chunking, retrieval).
- **Schema placement (regra 7):** Pydantic request/response schemas in `src/app/api/`. Internal cross-layer DTOs in `src/domain/{ctx}/dtos/` as dataclasses.
- **Idempotent jobs (regra 8):** `Job.action()` must tolerate re-execution.
- **UUID v7 PK** via `HasUUID`; timestamps via `HasTimestamps`.
- **Embeddings:** OpenAI `text-embedding-3-small`, dim 1536, configurable via settings. Decoupled from chat provider.
- **New deps approved by ADR-0007/0008/0009:** `pydantic-ai`, `pgvector`, `tavily-python`. No other new deps without discussion (regra 10).
- **Package manager:** UV. Add deps with `uv add`, run with `uv run`.
- **Tests:** external clients (Notion/MCP, LLM, embeddings, Tavily) are MOCKED/FAKED — never called for real. Repositories use a real Postgres (pgvector) via testcontainers. Fakes for domain, mocks for infra.

```python
        pkg = f"src.domain.{ctx.name}.models"
```

```python
"""Document — entidade de domínio (metadados da página aprovada). Pura."""

from dataclasses import dataclass
from datetime import datetime
from uuid import UUID


@dataclass
class Document:
    uuid: UUID
    notion_page_id: str
    title: str
    source_url: str
    status: str  # "approved" | "pending" | "archived"
    last_edited_time: datetime | None
    created_at: datetime
    updated_at: datetime
    deleted_at: datetime | None = None

    def is_approved(self) -> bool:
        return self.status == "approved" and self.deleted_at is None

    def is_indexable(self) -> bool:
        return self.is_approved()
```

```python
    async def get_by_id(self, document_id: UUID, with_trashed: bool = False) -> Document | None:
        stmt = select(DocumentModel).where(DocumentModel.uuid == document_id)
        if not with_trashed:
            stmt = stmt.where(DocumentModel.deleted_at.is_(None))
        model = (await self.session.execute(stmt)).scalar_one_or_none()
        return DocumentMapper.to_entity(model) if model else None

    async def get_by_notion_page_id(self, page_id: str) -> Document | None:
        stmt = select(DocumentModel).where(DocumentModel.notion_page_id == page_id)
        model = (await self.session.execute(stmt)).scalar_one_or_none()
        return DocumentMapper.to_entity(model) if model else None

    async def list_approved(self) -> list[Document]:
        stmt = select(DocumentModel).where(
            DocumentModel.status == "approved", DocumentModel.deleted_at.is_(None)
        )
        models = (await self.session.execute(stmt)).scalars().all()
        return [DocumentMapper.to_entity(m) for m in models]

    async def upsert_by_notion_page_id(self, entity: Document) -> Document:
        stmt = select(DocumentModel).where(
            DocumentModel.notion_page_id == entity.notion_page_id
        )
        model = (await self.session.execute(stmt)).scalar_one_or_none()
        attrs = DocumentMapper.to_model_attrs(entity)
        if model is None:
            model = DocumentModel(**attrs)
            self.session.add(model)
        else:
            for key, value in attrs.items():
                if key not in ("uuid", "notion_page_id"):
                    setattr(model, key, value)
        await self.session.flush()
        await self.session.refresh(model)
        return DocumentMapper.to_entity(model)

    async def soft_delete(self, document_id: UUID) -> None:
        from datetime import datetime, timezone

        model = (
            await self.session.execute(
                select(DocumentModel).where(DocumentModel.uuid == document_id)
            )
        ).scalar_one_or_none()
        if model is not None:
            model.deleted_at = datetime.now(timezone.utc)
            await self.session.flush()
```

```python
from abc import ABC, abstractmethod

from src.support.core.settings import settings


class EmbeddingsClient(ABC):
    @abstractmethod
    async def embed(self, texts: list[str]) -> list[list[float]]: ...

    async def embed_query(self, text: str) -> list[float]:
        return (await self.embed([text]))[0]


class OpenAIEmbeddingsClient(EmbeddingsClient):
    def __init__(self) -> None:
        from openai import AsyncOpenAI

        self._client = AsyncOpenAI(api_key=settings.OPENAI_API_KEY)
        self._model = settings.EMBEDDING_MODEL

    async def embed(self, texts: list[str]) -> list[list[float]]:
        if not texts:
            return []
        response = await self._client.embeddings.create(model=self._model, input=texts)
        return [item.embedding for item in response.data]


def get_embeddings_client() -> EmbeddingsClient:
    # Só OpenAI por ora (ver settings.EMBEDDING_PROVIDER).
    return OpenAIEmbeddingsClient()
```

```python
"""ChunkingService — divide texto em chunks com overlap. Puro, sem I/O."""

from src.support.core.settings import settings


class ChunkingService:
    def __init__(self, size: int | None = None, overlap: int | None = None) -> None:
        self.size = size or settings.RAG_CHUNK_SIZE
        self.overlap = overlap if overlap is not None else settings.RAG_CHUNK_OVERLAP
        if self.overlap >= self.size:
            raise ValueError("overlap deve ser menor que size")

    def split(self, text: str) -> list[str]:
        text = (text or "").strip()
        if not text:
            return []
        if len(text) <= self.size:
            return [text]

        step = self.size - self.overlap
        chunks: list[str] = []
        start = 0
        while start < len(text):
            chunks.append(text[start : start + self.size])
            start += step
        return chunks
```

```python
"""SyncFromNotionAction — ingere/atualiza a base a partir de páginas aprovadas do Notion.

Idempotente: upsert por notion_page_id, re-chunk + re-embed, e remoção (soft-delete)
de documentos que não estão mais aprovados. Ver ADR-0008.
"""

from dataclasses import dataclass
from datetime import datetime, timezone
from uuid import uuid7

from src.domain.documents.entities.document import Document
from src.domain.documents.entities.document_chunk import DocumentChunk
from src.domain.documents.repositories.document_chunk_repository import DocumentChunkRepository
from src.domain.documents.repositories.document_repository import DocumentRepository
from src.domain.documents.services.chunking_service import ChunkingService
from src.support.clients.embeddings.embeddings_client import EmbeddingsClient


@dataclass
class SyncResult:
    synced: int
    removed: int


class SyncFromNotionAction:
    def __init__(self, notion_client, embeddings_client: EmbeddingsClient) -> None:
        self.notion = notion_client
        self.embeddings = embeddings_client
        self.documents = DocumentRepository()
        self.chunks = DocumentChunkRepository()
        self.chunker = ChunkingService()

    async def execute(self) -> SyncResult:
        pages = await self.notion.list_approved_pages()
        approved_ids = {p.id for p in pages}

        synced = 0
        for page in pages:
            document = await self._upsert_document(page)
            await self._reindex_chunks(document, page.content)
            synced += 1

        removed = await self._remove_unapproved(approved_ids)
        return SyncResult(synced=synced, removed=removed)

    async def _upsert_document(self, page) -> Document:
        now = datetime.now(timezone.utc)
        entity = Document(
            uuid=uuid7(),
            notion_page_id=page.id,
            title=page.title,
            source_url=page.url,
            status="approved",
            last_edited_time=page.last_edited_time,
            created_at=now,
            updated_at=now,
        )
        return await self.documents.upsert_by_notion_page_id(entity)

    async def _reindex_chunks(self, document: Document, content: str) -> None:
        texts = self.chunker.split(content)
        vectors = await self.embeddings.embed(texts)
        chunks = [
            DocumentChunk(
                uuid=uuid7(),
                document_uuid=document.uuid,
                chunk_index=i,
                content=text,
                embedding=vector,
            )
            for i, (text, vector) in enumerate(zip(texts, vectors))
        ]
        await self.chunks.replace_for_document(document.uuid, chunks)

    async def _remove_unapproved(self, approved_ids: set[str]) -> int:
        removed = 0
        for document in await self.documents.list_approved():
            if document.notion_page_id not in approved_ids:
                await self.chunks.delete_for_document(document.uuid)
                await self.documents.soft_delete(document.uuid)
                removed += 1
        return removed
```

```python
"""KnowledgeRetrievalService — busca semântica top-k no pgvector (só aprovados).

Ver ADR-0008 (RAG híbrido). Retorna chunks com metadados de fonte para citação.
"""

from sqlalchemy import select

from src.domain.documents.dtos.retrieved_chunk import RetrievedChunk
from src.domain.documents.models.document import DocumentModel
from src.domain.documents.models.document_chunk import DocumentChunkModel
from src.support.clients.embeddings.embeddings_client import EmbeddingsClient
from src.support.core.context import CurrentAsyncSessionContext
from src.support.core.settings import settings


class KnowledgeRetrievalService:
    def __init__(self, embeddings_client: EmbeddingsClient, top_k: int | None = None) -> None:
        self.embeddings = embeddings_client
        self.top_k = top_k or settings.RAG_TOP_K
        self.session = CurrentAsyncSessionContext.get()

    async def search(self, query: str) -> list[RetrievedChunk]:
        query_vec = await self.embeddings.embed_query(query)
        distance = DocumentChunkModel.embedding.cosine_distance(query_vec)

        stmt = (
            select(
                DocumentChunkModel.content,
                DocumentModel.notion_page_id,
                DocumentModel.title,
                DocumentModel.source_url,
                distance.label("distance"),
            )
            .join(DocumentModel, DocumentChunkModel.document_uuid == DocumentModel.uuid)
            .where(
                DocumentModel.status == "approved",
                DocumentModel.deleted_at.is_(None),
```

```python
        """Busca na base de conhecimento aprovada (Notion). Use para regras do ecossistema."""
        chunks = await ctx.deps.retrieval.search(query)
        for c in chunks:
            ctx.deps.citations.append(

                    source_type="notion",
                    title=c.title,
                    url=c.source_url,
                    snippet=c.content[:280],
                    page_id=c.notion_page_id,

                     snippet=page.content[:280], page_id=page.id)

    async def web_search(ctx: RunContext[OracleDeps], query: str) -> str:
        """Busca na web (Tavily). Use só quando a base interna não cobrir."""

                Citation(source_type="web", title=r.title, url=r.url, snippet=r.content[:280])
            )

            return _wrap("Sem resultados web.")
        return _wrap("\n\n".join(f"[{r.title}] {r.url}\n{r.content}" for r in results))

    return agent
```

```python
"""OracleEngine — fronteira consumida pelo domínio. Encapsula o Agent Pydantic AI."""

from typing import AsyncIterator

from src.domain.shared.value_objects.citation import Citation
from src.support.agent.oracle_agent import OracleDeps, build_agent
from src.support.agent.ports import AgentMessage, AgentStreamChunk


def _history_preamble(history: list[AgentMessage]) -> str:
    if not history:
        return ""
    lines = [f"{m.role}: {m.content}" for m in history[-10:]]
    return "Histórico recente da conversa:\n" + "\n".join(lines) + "\n\n"


def _dedup(citations: list[Citation]) -> list[Citation]:
    seen: set[tuple] = set()
    out: list[Citation] = []
    for c in citations:
        key = (c.source_type, c.url, c.page_id)
        if key not in seen:
            seen.add(key)
            out.append(c)
    return out


class OracleEngine:
    def __init__(self, retrieval, notion, tavily) -> None:
        self._agent = build_agent()
        self._retrieval = retrieval
        self._notion = notion
        self._tavily = tavily

    async def stream_answer(
        self, question: str, history: list[AgentMessage]
    ) -> AsyncIterator[AgentStreamChunk]:
        deps = OracleDeps(retrieval=self._retrieval, notion=self._notion, tavily=self._tavily)
        prompt = _history_preamble(history) + f"Pergunta do usuário: {question}"

        async with self._agent.run_stream(prompt, deps=deps) as result:
            async for delta in result.stream_text(delta=True):
                yield AgentStreamChunk(type="text", text=delta)

        yield AgentStreamChunk(type="sources", citations=_dedup(deps.citations))
```

```markdown
# ADR-0010 — DBSessionMiddleware como middleware ASGI puro (streaming-safe)

## Status

Aceito — 2026-07-03.

## Resumo

- **Decisão:** o `DBSessionMiddleware` passa a ser um middleware **ASGI puro**
  (não `BaseHTTPMiddleware`), abrindo/fechando a sessão async por request e
  comitando após o corpo da resposta ser totalmente enviado.
- **Aplica-se quando:** mexer no ciclo de vida da sessão de banco por request,
  ou em endpoints de streaming (SSE).
- **Regra prática:** a sessão de request continua vindo do contexto (regra 3);
  agora ela sobrevive ao streaming SSE porque o middleware não bufferiza a resposta.

---

## Contexto

O chat do oráculo transmite a resposta via SSE (ADR-0009). O `DBSessionMiddleware`
original herdava de `starlette.middleware.base.BaseHTTPMiddleware`, que **bufferiza
o corpo inteiro** da resposta antes de enviá-lo — incompatível com streaming
incremental. Além disso, a sessão precisa continuar aberta enquanto o gerador SSE
persiste a mensagem do assistente ao final.

## Decisão

Reescrever o middleware como ASGI puro: abre `AsyncSessionLocal()`, popula
`CurrentAsyncSessionContext`, intercepta `send` para detectar o fim do corpo
(`more_body=False`) e então comita; em exceção, faz rollback. Mantém a criação
lazy da sessão sync (jobstore/scripts) como antes.

## Consequências

**Positivas**
- SSE funciona com token-a-token; a sessão de request continua válida durante o stream.
- Mantém a regra 3 (sessão via contexto) para todo o app.

**Negativas / riscos**
- A sessão fica aberta por toda a duração do stream (aceitável no MVP).
- Middleware ASGI é mais baixo nível que `BaseHTTPMiddleware` — exige cuidado com
  a ordem de `send`/rollback.

## Alternativas consideradas

- **Sessão própria no gerador SSE:** rejeitada — violaria a regra 3 pontualmente e
  espalharia gerência de sessão pelo domínio.
- **Bufferizar a resposta do agente:** rejeitada — perde a UX de streaming (ADR-0009).
```

```python
"""Middleware ASGI puro: abre a sessão async por request, comita ao fim do corpo.

Ver ADR-0010. Não bufferiza a resposta — compatível com SSE.
"""

from src.support.core.context import CurrentAsyncSessionContext, CurrentSessionContext
from src.support.core.database import AsyncSessionLocal, SessionLocal


class DBSessionMiddleware:
    def __init__(self, app) -> None:
        self.app = app

    async def __call__(self, scope, receive, send) -> None:
        if scope["type"] != "http":
            await self.app(scope, receive, send)
            return

        async with AsyncSessionLocal() as async_session:
            CurrentAsyncSessionContext.set(async_session)
            CurrentSessionContext.set_factory(SessionLocal)

            failed = False

            async def send_wrapper(message) -> None:
                if message["type"] == "http.response.body" and not message.get("more_body", False):
                    # corpo totalmente enviado → comita antes de fechar
                    if not failed:
                        await async_session.commit()
                        sync_session = _peek_sync()
                        if sync_session is not None:
                            sync_session.commit()
                await send(message)

            try:
                await self.app(scope, receive, send_wrapper)
            except Exception:
                failed = True
                await async_session.rollback()
                raise
            finally:
                sync_session = _peek_sync()
                if sync_session is not None:
                    sync_session.close()
                CurrentSessionContext.clear()
                CurrentAsyncSessionContext.clear()


def _peek_sync():
    from src.support.core.context.current_db_session_context import _current_session

    return _current_session.get()
```

```python
"""AnswerQuestionAction — orquestra: histórico → motor → persiste turno (streaming)."""

from datetime import datetime, timezone
from typing import AsyncIterator
from uuid import UUID

from uuid6 import uuid7

from src.domain.conversations.entities.message import Message
from src.domain.conversations.enums.message_role import MessageRole
from src.domain.conversations.repositories.conversation_repository import ConversationRepository
from src.domain.conversations.repositories.message_repository import MessageRepository
from src.support.agent.ports import AgentMessage, AgentStreamChunk
from src.support.core.exceptions import NotFoundError


class AnswerQuestionAction:
    def __init__(self, engine) -> None:
        self.engine = engine
        self.conversations = ConversationRepository()
        self.messages = MessageRepository()

    async def execute(
        self, conversation_id: UUID, question: str
    ) -> AsyncIterator[AgentStreamChunk]:
        conversation = await self.conversations.get_by_id(conversation_id)
        if conversation is None:
            raise NotFoundError("Conversa não encontrada.")

        history = await self.messages.list_for_conversation(conversation_id)
        agent_history = [AgentMessage(role=m.role, content=m.content) for m in history]

        await self._persist(conversation_id, MessageRole.USER.value, question, [])

        answer_parts: list[str] = []
        citations = []
        async for chunk in self.engine.stream_answer(question, agent_history):
            if chunk.type == "text":
                answer_parts.append(chunk.text)
            elif chunk.type == "sources":
                citations = chunk.citations
            yield chunk

        await self._persist(
            conversation_id, MessageRole.ASSISTANT.value, "".join(answer_parts), citations
        )

    async def _persist(self, conversation_id, role, content, citations) -> None:
        await self.messages.add(
            Message(
                uuid=uuid7(),
                conversation_uuid=conversation_id,
                role=role,
                content=content,
                created_at=datetime.now(timezone.utc),
                citations=citations,
            )
        )
```

```python
import json
from uuid import UUID

from fastapi import Request
from fastapi.responses import StreamingResponse

from src.app.api.requests.conversations import AskQuestionRequest, CreateConversationRequest
from src.app.api.responses.conversations import ConversationDetailResponse, ConversationResponse
from src.domain.conversations.actions.answer_question_action import AnswerQuestionAction
from src.domain.conversations.actions.create_conversation_action import CreateConversationAction
from src.domain.conversations.actions.get_conversation_action import GetConversationAction
from src.domain.documents.services.knowledge_retrieval_service import KnowledgeRetrievalService
from src.support.agent.oracle_engine import OracleEngine
from src.app.api.responses.conversations import CitationResponse
from src.support.clients.embeddings.embeddings_client import get_embeddings_client
from src.support.clients.notion.notion_client import NotionClient
from src.support.clients.tavily.tavily_client import TavilyClient


def _build_engine() -> OracleEngine:
    embeddings = get_embeddings_client()
    return OracleEngine(
        retrieval=KnowledgeRetrievalService(embeddings_client=embeddings),
        notion=NotionClient(),
        tavily=TavilyClient(),
    )


class ConversationController:
    @staticmethod
    async def create(data: CreateConversationRequest) -> ConversationResponse:
        conversation = await CreateConversationAction().execute(title=data.title)
        return ConversationResponse.from_entity(conversation)

    @staticmethod
    async def get(conversation_id: UUID) -> ConversationDetailResponse:
        conversation, messages = await GetConversationAction().execute(conversation_id)
        return ConversationDetailResponse.from_entities(conversation, messages)

    @staticmethod
    async def ask(conversation_id: UUID, data: AskQuestionRequest, request: Request) -> StreamingResponse:
        action = AnswerQuestionAction(engine=_build_engine())

        async def event_stream():
            async for chunk in action.execute(conversation_id, data.question):
                if chunk.type == "text":
                    yield f"event: token\ndata: {json.dumps(chunk.text)}\n\n"
                elif chunk.type == "sources":
                    payload = [CitationResponse.from_vo(c).model_dump() for c in chunk.citations]
                    yield f"event: sources\ndata: {json.dumps(payload)}\n\n"
            yield "event: done\ndata: {}\n\n"

        return StreamingResponse(event_stream(), media_type="text/event-stream")
```

`NotFoundError` raised inside `action.execute` before streaming starts is translated by the existing `exception_handlers`. (If the generator raises after the response has begun, the stream simply ends — acceptable for MVP; a mid-stream error event is fast-follow.)
